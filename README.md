# DefiBot

Chart-first DeFi paper trading and backtesting platform inspired by Jesse and Freqtrade.

## Current scope

- Next.js research terminal with interactive candlestick chart shell
- mock order book and market overview API
- venue and strategy research for a small-balance starting point
- product direction focused on realistic paper trading before live execution

## Strategy priorities

1. Micro Reversion Maker
2. Trend Breakout Perp
3. Funding Basis Carry

## Commands

```bash
npm install
npm run typecheck
npm run build
```
