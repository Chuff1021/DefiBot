DefiBot

This repository is a TypeScript foundation for a personal-use DeFi trading bot. It is now deployable on Vercel with a static homepage and a lightweight API status endpoint.

What This Repo Contains

  • Shared domain types for chains, venues, execution, risk, and portfolio state.
  • Configuration registries for initial chains, venues, and strategy families.
  • Scaffold module boundaries for data, indicators, strategies, execution, risk, and portfolio.
  • A simple Vercel deployment surface at / and /api/status.
